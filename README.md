# blinks-mobile-sdk

A sdk for rendering blinks in your react native apps

## Installation

```sh
npm install blinks-mobile-sdk
```

## Usage


```js
import { multiply } from 'blinks-mobile-sdk';

// ...

const result = await multiply(3, 7);
```
